# 🧬 BioHub 

Интерактивная веб-платформа, посвященная биоинженерии, генетике и технологиям будущего. Проект разработан как демонстрационное веб-приложение для популяризации науки и визуализации сложных молекулярно-биологических процессов.

## Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone (https://sonchiik-ux.github.io/BioHub/)
   ```
2. Откройте файл `index.html` в любом современном браузере.
